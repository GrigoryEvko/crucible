# scripts/mint-pattern-allowlist.txt — §XXI scanner grandfather list.
#
# Each entry is `path:line` (exempts all four checks) OR
# `path:line:<check>-ok` where <check> ∈ {nodiscard, noexcept,
# constexpr, requires} (exempts ONE check).  Lines beginning with
# '#' are comments.
#
# Trim discipline: every entry is either a structural-exempt
# annotation (passkey constructor, allocating mint, class method,
# friend-forward edge case) OR a TODO awaiting migration.  When
# the listed site migrates to the canonical §XXI shape (or the
# regex learns to recognize a new structural false-positive), the
# allowlist line MUST be removed; the goal is for this file to
# trend toward the structural-exempt minimum.
#
# Naming convention for the TODO half:
#   `requires-ok` on a templated `mint_*` whose gate is
#     expressed as `static_assert(...)` in the body rather than a
#     `requires` clause on the signature.  §XXI rule (4) demands
#     a `requires` clause.  Migration shape: lift the in-body
#     static_assert to a concept and add the requires clause.
#
#   `constexpr-ok` on an allocating mint or class-method mint
#     whose factory cannot be `constexpr` for structural reasons
#     (returns a ScopedView, allocates a runtime channel, threads
#     a non-constexpr resource).  §XXI rule (3) explicitly permits
#     dropping `constexpr` "in favour of plain [[nodiscard]]
#     noexcept" for genuinely-allocating mints.
#
#   `noexcept-ok` on a mint that legitimately throws OR is being
#     migrated.  §XXI rule (2) demands noexcept of every mint
#     (even allocating ones) — so this category is mostly
#     migration-TODO, not structural-exempt.
#
#   `nodiscard-ok` on a mint whose return value can legitimately
#     be discarded (e.g. an inheritance-thread mint that mutates
#     ambient state).  §XXI rule (1) demands [[nodiscard]] on
#     every mint, so this category is also mostly migration-TODO.
#

#   fixy-M-23 (#1508) tracks mint_persisted_session noexcept drift.
#   fixy-A4-027 (#1666) tracks mint_federation_admittance nodiscard drift.
